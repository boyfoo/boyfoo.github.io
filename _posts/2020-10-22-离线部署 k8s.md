##### 关闭交换分区

* centos

```bash
# 临时关闭
$ swapoff -a
```
```bash
# 永久关闭 注释文件内容
$ vim /etc/fstab
#/swapfile none swap defaults 0 0
```
```bash
#查看关闭结果
$ free -m
# Swap 都为0表示关闭成功
Swap:             0           0           0
```

* centos

